Connecting to BigQuery

Here’s how to create a BigQuery “Service Account” and add it as a connection in Wagon.

You must be an Owner on your Google Cloud project.

  1. Go to Google Cloud console and select your project.

  2. Click Create credentials then Service account key.

  3. Choose Service account then “New service account” and name it.

  4. Select Key type then JSON and click Create.

  5. Download the .json file, open it in a text editor, and copy the full JSON string into the Wagon connection modal.

  6. Click connect and run this example query. (Open query in Wagon):

     select
   name,
   year,
   count(*)
 from [bigquery-public-data:usa_names.usa_1910_2013]
 where name in ('Homer', 'Marge', 'Lisa', 'Maggie', 'Bart', 'Bort')
 group by name, year;

If you have trouble, message us in the Wagon app or email support@wagonhq.com.

For more detail, read Google’s official instructions.
